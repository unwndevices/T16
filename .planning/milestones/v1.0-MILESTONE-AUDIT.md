## Integration Issues

### Critical: CC Per-Parameter Sync
Web `sendParamUpdate` sends 4 payload bytes for `DOMAIN_BANK_CC`. Firmware `HandleParamSet` expects 5 bytes and rejects shorter payloads. CC mapping edits from the web editor are silently dropped. **Fix:** Add dedicated CC send function or adjust firmware handler.

### Important: Schema 'pal' Field
Firmware serializes `"pal"` (palette) per bank. JSON schema has `additionalProperties: false`. Export-then-reimport fails. **Fix:** Add `"pal"` to schema and types, or strip during parsing.

### Known: BLE MIDI Bridging
BLE connects via Web Bluetooth but `input`/`output` remain null. Config operations require non-null MIDI I/O. **Blocked by:** firmware SysEx chunking over BLE MTU.
