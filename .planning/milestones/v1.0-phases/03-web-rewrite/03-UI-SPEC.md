# Phase 3 — UI Design Contract

> Visual and interaction contract for the web configurator rewrite. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (custom design system) |
| Preset | not applicable |
| Component library | Radix UI primitives (@radix-ui/*) |
| Icon library | react-icons (Material Design subset: md prefix) |
| Font | Inter (body), Poppins (headings) via @fontsource |

Source: D-01, D-03 from CONTEXT.md. react-icons carried forward from existing codebase.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline element padding |
| sm | 8px | Compact card padding, list item gaps |
| md | 16px | Default element spacing, card body padding |
| lg | 24px | Section padding, tab panel padding |
| xl | 32px | Layout column gaps |
| 2xl | 48px | Page horizontal padding |
| 3xl | 64px | Page top/bottom margins |

Exceptions: Touch targets for nav icon buttons use 44px minimum (accessibility).

CSS custom property format:
```css
--space-xs: 4px;
--space-sm: 8px;
--space-md: 16px;
--space-lg: 24px;
--space-xl: 32px;
--space-2xl: 48px;
--space-3xl: 64px;
```

## Typography

| Role | Size | Weight | Line Height | Font Family |
|------|------|--------|-------------|-------------|
| Body | 14px (0.875rem) | 400 (regular) | 1.5 | Inter |
| Label | 12px (0.75rem) | 600 (semibold) | 1.4 | Inter |
| Heading | 20px (1.25rem) | 600 (semibold) | 1.2 | Poppins |
| Page Title | 28px (1.75rem) | 600 (semibold) | 1.2 | Poppins |

Only two weights used across the entire UI: 400 (regular) for body text, 600 (semibold) for labels and headings.

CSS custom property format:
```css
--font-body: 'Inter', sans-serif;
--font-heading: 'Poppins', sans-serif;
--font-mono: 'Menlo', monospace;
--text-xs: 0.75rem;
--text-sm: 0.875rem;
--text-md: 1.25rem;
--text-lg: 1.75rem;
--weight-regular: 400;
--weight-semibold: 600;
```

Source: D-03 from CONTEXT.md. Sizes derived from existing Chakra theme (sm, md, xl, 2xl mapped to the 4 roles).

## Color

Colors are converted from the existing Chakra theme palettes (D-04) to CSS custom properties.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--gray-50` (#faf9fe) | Page background |
| Secondary (30%) | `--gray-800` (#202022) nav, `--white` (#ffffff) cards | Navigation bar background, card surfaces |
| Accent (10%) | `--primary-800` (#6246ea) | Primary CTA buttons, active nav indicator, sync status dot (synced), focused input borders |
| Destructive | `--red-500` (#e4412c) | Calibration reset button, disconnect state dot, error toasts |

### Full palette (CSS custom properties)

```css
/* Neutrals */
--gray-50: #faf9fe;
--gray-100: #f2f0fd;
--gray-200: #e8e6f3;
--gray-300: #d4d2df;
--gray-400: #adacb6;
--gray-500: #7f7e85;
--gray-600: #555459;
--gray-700: #373639;
--gray-800: #202022;
--gray-900: #19191b;
--black: #100f10;

/* Primary (purple) */
--primary-500: #917df0;
--primary-600: #816bee;
--primary-700: #7258ec;
--primary-800: #6246ea;
--primary-900: #5234e8;

/* Secondary (pink) — reserved for brand flourishes only */
--secondary-500: #f770df;
--secondary-800: #f328cf;

/* Accent (yellow-green) — NOT used as interactive accent */
--accent-500: #e5f10a;

/* Semantic */
--color-success: #2096a6;  /* teal-500, connection active */
--color-error: #e4412c;    /* red-500, errors and destructive */
--color-warning: #eaf51a;  /* accent-400, unsaved changes */
```

Accent reserved for: primary CTA buttons, active navigation indicator, focused input rings, synced-state status dot.

Note: The `accent` palette (yellow-green) from the existing theme is kept as a token but NOT used as the interactive accent color. The `primary` palette (purple) serves as the interactive accent, consistent with the existing UI's visual hierarchy.

## Component Inventory

Design system primitives to build with Radix UI (D-01):

| Component | Radix Primitive | Purpose |
|-----------|----------------|---------|
| Button | n/a (custom) | CTA, nav actions, destructive actions |
| Card | n/a (custom) | Setting groups (SelectCard, SliderCard, ToggleCard, NumberCard) |
| Select | @radix-ui/react-select | Dropdown selectors (mode, scale, channel) |
| Switch | @radix-ui/react-switch | Boolean toggles (demo mode, MIDI TRS, BLE) |
| Slider | @radix-ui/react-slider | Numeric range inputs (velocity curve, CC values) |
| Tabs | @radix-ui/react-tabs | Dashboard page sections (Keyboard, Scales, CC, Settings) |
| Dialog | @radix-ui/react-dialog | Confirmations, calibration prompt |
| AlertDialog | @radix-ui/react-alert-dialog | Destructive action confirmations |
| Tooltip | @radix-ui/react-tooltip | Icon button labels, parameter descriptions |
| NavigationMenu | @radix-ui/react-navigation-menu | Top nav bar |
| Toast | @radix-ui/react-toast | Connection feedback, save confirmation, errors |
| Skeleton | n/a (custom) | Loading state while device not connected |

### Interaction States

All interactive components must implement these states:

| State | Visual Treatment |
|-------|-----------------|
| Default | `--gray-200` border, `--gray-50` background |
| Hover | `--gray-300` border, `--gray-100` background |
| Focus | 2px `--primary-800` ring, 2px offset |
| Active/Pressed | `--primary-900` background on buttons |
| Disabled | 50% opacity, cursor not-allowed |
| Loading/Skeleton | `--gray-200` pulsing background, border-radius: 8px |

### Transitions

All state transitions use: `150ms ease-out`. No transition on layout shifts. Page transitions: none (instant route change).

## Layout Contract

### Responsive Breakpoints

| Name | Min Width | Layout Change |
|------|-----------|---------------|
| mobile | 0px | Single column, stacked cards |
| tablet | 768px | Two-column card grid |
| desktop | 1024px | Full layout, side-by-side panels |

Source: Claude's discretion per CONTEXT.md.

### Page Structure

```
+--------------------------------------------------+
| NavBar (sticky top, 56px height)                 |
|  [Editor] [Upload] [Manual]  Title    [Status]   |
+--------------------------------------------------+
| Page Content (max-width: 960px, centered)        |
|                                                   |
|  Tabs / Page-specific content                    |
|                                                   |
+--------------------------------------------------+
| Footer                                           |
+--------------------------------------------------+
```

- NavBar: sticky, `--gray-800` background on dark variant or `rgba(255,255,255,0.85)` with backdrop-blur on light
- Content area: max-width 960px, centered, horizontal padding `--space-2xl` (48px) on desktop, `--space-md` (16px) on mobile
- Footer: static, `--gray-100` background

### Sync Status Indicator

| State | Visual | Label |
|-------|--------|-------|
| Connected + synced | `--color-success` filled dot (8px) | none |
| Connected + unsynced | `--color-warning` filled dot (8px) | none |
| Disconnected | `--color-error` filled dot (8px) | none |

Per-parameter sync: each card shows a small dot (6px) next to its label when its value differs from device state. Dot color: `--color-warning`.
