    **Create `editor-tx/src/services/ble.ts`:**

    Export constants:
    ```typescript
    export const BLE_MIDI_SERVICE = '03b80e5a-ede8-4b33-a751-6ce34ec4c700'
    export const BLE_MIDI_CHAR = '7772e5db-3868-4112-a1a9-f2669d106bf3'
    ```

    Export `parseBLEMidiPacket(data: DataView): Uint8Array[]`:
    - BLE MIDI packet format: [header_byte] [timestamp_low midi_status midi_data...] [timestamp_low midi_status midi_data...]
    - Header byte has bit 7 set
    - Timestamp low bytes have bit 7 set
    - MIDI status bytes have bit 7 set, but they are preceded by a timestamp byte
    - Implementation: skip header (byte 0). Iterate from byte 1. When byte has bit 7 set, it's a timestamp -- skip it. When next byte also has bit 7 set and is a MIDI status (0x80-0xEF), start collecting a new message. Collect data bytes (bit 7 clear) until next timestamp or end.
    - Return array of Uint8Array, each being a complete MIDI message.

    Export `connectBLE(): Promise<{ characteristic: BluetoothRemoteGATTCharacteristic; device: BluetoothDevice }>`:
    - Call `navigator.bluetooth.requestDevice({ filters: [{ name: 'Topo T16' }], optionalServices: [BLE_MIDI_SERVICE] })`
    - Connect to GATT server: `device.gatt!.connect()`
    - Get primary service: `server.getPrimaryService(BLE_MIDI_SERVICE)`
    - Get characteristic: `service.getCharacteristic(BLE_MIDI_CHAR)`
    - Start notifications: `characteristic.startNotifications()`
    - Return { characteristic, device }
