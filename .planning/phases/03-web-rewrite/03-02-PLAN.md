Build each component following this pattern (per D-01, D-02):
- Each component: `ComponentName.tsx` (named export), `ComponentName.module.css` (CSS Module consuming tokens), `index.ts` (barrel)
- Import Radix primitives from `radix-ui` (unified package)
- All CSS uses `var(--token)` references to tokens.css
- All interactive states per UI-SPEC Interaction States table: default (--gray-200 border, --gray-50 bg), hover (--gray-300 border, --gray-100 bg), focus (2px --primary-800 ring, 2px offset), active (--primary-900 bg on buttons), disabled (50% opacity, cursor not-allowed)
- All transitions: 150ms ease-out (use var(--transition-fast))
- All border-radius: var(--radius-md) (8px) unless specified otherwise
- Accept `ref` as a regular prop (React 19, no forwardRef)
- Use named exports only (no default exports)

**Component specifications:**

1. **Button** (custom, no Radix): Props: `variant: 'primary' | 'secondary' | 'destructive' | 'ghost'`, `size: 'sm' | 'md'`, `disabled`, `loading`, `children`, `onClick`, standard button HTML attrs. Primary variant: --primary-800 bg, white text. Destructive: --color-error bg, white text. Ghost: transparent bg, --gray-600 text.

2. **Card** (custom, no Radix): Props: `children`, `className`. Surface container with --white bg, 1px --gray-200 border, --radius-md, padding var(--space-md), optional title slot.

3. **Select**: Wraps `radix-ui` Select. Props: `label`, `value`, `onValueChange`, `options: {value: string, label: string}[]`, `disabled`. Follow RESEARCH.md Pattern 2 exactly.

4. **Switch**: Wraps `radix-ui` Switch. Props: `label`, `checked`, `onCheckedChange`, `disabled`. Thumb slides left/right. Checked: --primary-800 bg. Unchecked: --gray-300 bg.

5. **Slider**: Wraps `radix-ui` Slider. Props: `label`, `value`, `onValueChange`, `min`, `max`, `step`, `disabled`. Track: --gray-200. Range fill: --primary-800. Thumb: white with --gray-300 border.

6. **Tabs**: Wraps `radix-ui` Tabs. Export `Tabs` (Root), `TabsList`, `TabsTrigger`, `TabsContent`. Active trigger: --primary-800 bottom border. Inactive: --gray-500 text.

7. **Dialog**: Wraps `radix-ui` Dialog. Export `Dialog` (Root), `DialogTrigger`, `DialogContent`, `DialogTitle`, `DialogDescription`, `DialogClose`. Overlay: rgba(0,0,0,0.4) with backdrop-blur. Content: white bg, --radius-lg, max-width 480px.

8. **Toast**: Wraps `radix-ui` Toast. Export `ToastProvider`, `ToastViewport`, `Toast`, `ToastTitle`, `ToastDescription`, `ToastClose`. Position: bottom-right. Variants: default, error (--color-error left border), success (--color-success left border).

9. **Tooltip**: Wraps `radix-ui` Tooltip. Export `TooltipProvider`, `Tooltip`, `TooltipTrigger`, `TooltipContent`. Dark bg (--gray-800), white text, --text-xs, --radius-sm, padding --space-xs --space-sm.

10. **Skeleton** (custom, no Radix): Props: `width`, `height`, `borderRadius`. Pulsing --gray-200 background animation.
